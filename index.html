gallery
  • git status - посмотреть изменения в репозитории
  • git add . - собираетесь сделать изменения в следующем коммите
  • git commit -m"add" - фиксирование изменения в проекте
  • git remote -v - для получения удаленных веток и дополнительной информации
  • git push origin main - отправка работы в хранилище
  • git checkout -b создание новой ветки
  • git log - проверить историю
  • git pull origin - извлекает и загружает содержтмое из удаленного репозитория
