Списки Селекторов

по названию элемента, например section{padding-top и значение}

по классу - когда нужно выбрать стиль для одного или нескольких элементов, например .description {color}

по id -уникальное имя элемента для которого используются стили и тд. например #имя индентификатора {описание стиля}

по нескольким классам - для любого элемента одновременно можно добавить несколько символов, например -.class1.class2 { описание стиля }

по атрибутам - устанавливает стиль для элемента, если указан спецефический атрибут, например [атрибут] { описание правил стиля }

по значению атрибута - Устанавливает стиль элемента, если задан специфический атрибут, значение в этом случае не важно, напрмер [атрибут] { описание правил стиля }

начало название по атрибуту - устанавливает стиль элемента в том случае, если задано определенное значение специфического атрибута, напрмер [атрибут="значення"] { описание правил стиля }

по окончанию атрибута - устанавливает стиль элемента в том случае, если значение атрибута начинается с указанного текста, например [атрибут^="значення"] { описание правил стиля }

в любом месте атрибута - Возможны варианты, когда стиль применим к тегу определенным атрибутом, когда частью значения является некоторый текст. При этом точно не известно, в каком месте значение включен этот текст в начале, середине или конце, например [атрибут*="значение"] { описание правил стиля }

вложеные селекторы - При создании веб-страницы часто приходится вкладывать одни элементы внутрь других. Чтобы стили этих элементов использовались корректно, помогут вложенные селекторы, например E F { Опис правил стилю }
